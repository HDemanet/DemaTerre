Accessibilité

Notre engagement pour un site accessible à tous

Dernière mise à jour : Novembre 2025

DemaTerre s'engage à rendre son site web accessible à toutes et à tous, conformément aux standards d'accessibilité internationaux (WCAG 2.1).

Notre engagement

Nous croyons que l'accessibilité numérique est un droit fondamental. Nous nous efforçons de garantir que notre site soit utilisable par le plus grand nombre, quelles que soient les capacités physiques, sensorielles ou cognitives de chacun.

Conformité

Ce site vise à être conforme avec le niveau AA des WCAG 2.1 (Web Content Accessibility Guidelines). Cela signifie que nous nous efforçons de respecter les critères suivants :

  • Perceptible : L'information et les composants de l'interface doivent être présentés de manière à ce qu'ils puissent être perçus par tous
  • Utilisable : Les composants de l'interface et la navigation doivent être utilisables
  • Compréhensible : L'information et l'utilisation de l'interface doivent être compréhensibles
  • Robuste : Le contenu doit être suffisamment robuste pour être interprété de manière fiable par une large variété d'agents utilisateurs, y compris les technologies d'assistance

Mesures d'accessibilité mises en place

Structure et navigation

  • Utilisation de balises HTML sémantiques (header, nav, main, footer, section, article)
  • Hiérarchie de titres logique et cohérente
  • Lien d'évitement ("Aller au contenu principal") en début de page
  • Navigation au clavier fonctionnelle sur tous les éléments interactifs
  • Indicateurs de focus visibles
  • Attributs ARIA pour améliorer l'expérience avec les lecteurs d'écran

Contenu

  • Contraste des couleurs conforme aux normes AA
  • Taille de police adaptable et lisible
  • Textes alternatifs sur toutes les images importantes
  • Icônes décoratives masquées aux lecteurs d'écran (aria-hidden)
  • Vidéos et contenus multimédias avec alternatives textuelles

Formulaires

  • Labels explicites pour tous les champs de formulaire
  • Messages d'erreur clairs et accessibles
  • Instructions et indications d'aide disponibles
  • Champs obligatoires clairement identifiés

Responsive et adaptabilité

  • Site entièrement responsive (mobile, tablette, desktop)
  • Zoom jusqu'à 200% sans perte de contenu ou de fonctionnalité
  • Espacement suffisant entre les éléments interactifs (minimum 44x44 pixels)

Technologies utilisées

Ce site web repose sur les technologies suivantes :

  • HTML5
  • CSS3
  • JavaScript (avec dégradation gracieuse)

Le site est conçu pour fonctionner avec les technologies d'assistance courantes, notamment :

  • Lecteurs d'écran (NVDA, JAWS, VoiceOver, TalkBack)
  • Logiciels de grossissement
  • Logiciels de reconnaissance vocale
  • Navigation au clavier

Navigateurs et technologies d'assistance compatibles

Ce site a été testé pour fonctionner avec les navigateurs récents :

  • Google Chrome (dernières versions)
  • Mozilla Firefox (dernières versions)
  • Safari (dernières versions)
  • Microsoft Edge (dernières versions)

Compatibilité mobile :

  • iOS Safari
  • Android Chrome

Limitations connues

Malgré nos efforts, certaines parties du site peuvent présenter des limitations d'accessibilité :

  • Certaines images de fond décoratives peuvent ne pas avoir d'alternative textuelle (car elles sont décoratives)
  • Les contenus externes (cartes, vidéos embarquées) peuvent ne pas être entièrement sous notre contrôle

Nous travaillons continuellement à améliorer l'accessibilité de notre site.

Améliorations continues

L'accessibilité est un processus continu. Nous nous engageons à :

  • Effectuer des audits réguliers de l'accessibilité
  • Former notre équipe aux bonnes pratiques d'accessibilité
  • Intégrer l'accessibilité dès la conception de nouvelles fonctionnalités
  • Corriger les problèmes d'accessibilité signalés dans les plus brefs délais
