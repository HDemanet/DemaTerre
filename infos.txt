PHILOSOPHIE DE DESIGN :
======================

1. INSPIRATION TREE-GARDEN :
   - Hero sections avec images de fond
   - Cartes services avec hover effects
   - Palette verte naturelle + orange chaleureux
   - Typographie claire et espacement généreux
   - Photos de qualité (à ajouter)

2. COMPOSANTS RÉUTILISABLES :
   - Navbar/Footer dans des fichiers separes pour eviter de se repeter
   - Système de templates pour éviter la duplication
   - Classes CSS modulaires et cohérentes
   - Animations subtiles et fluides

3. ACCESSIBILITÉ INTÉGRÉE :
   - Pas de boutons/contrôles supplémentaires
   - Contrastes respectés dès la conception
   - Structure sémantique HTML5 native
   - Navigation clavier fluide
   - Textes alternatifs et ARIA intégrés

4. PERFORMANCE :
   - CSS organisé par couches logiques
   - JavaScript modulaire et lazy-loading
   - Images optimisées (WebP + fallback)
   - Critical CSS inline

5. COULEURS DEMATERRE :
   - Primary: #B55232 (Orange-brun terre cuite)
   - Secondary: #8EB36A (Vert tendre nature)
   - Tertiary: #44725C (Vert foncé forêt)
   - Neutrals: Gamme de gris ou beiges chauds

PROCHAINES ÉTAPES :
==================
1. Créer tous les fichiers HTML avec templates
2. Développer le système CSS modulaire
3. Implémenter les composants JavaScript
